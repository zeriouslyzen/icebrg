# Utility Scripts

This directory contains utility scripts for maintenance and development tasks.

## Scripts

### Python Scripts
- `comprehensive_audit.py` - Comprehensive codebase audit tool
- `audit_codebase.py` - Codebase auditing script

### Shell Scripts
- `check_env.sh` - Environment variable checker
- `create_uncensored_model.sh` - Creates uncensored Ollama model

## Usage

### Environment Check
```bash
./scripts/utilities/check_env.sh
```

### Create Uncensored Model
```bash
./scripts/utilities/create_uncensored_model.sh
```

### Run Audit
```bash
python scripts/utilities/comprehensive_audit.py
```
