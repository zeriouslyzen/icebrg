## üöÄ **Creating Your Own macOS Apps**

### **Method 1: Using ICEBURG CLI**
```bash
# Enable visual generation
export ICEBURG_ENABLE_VISUAL_GEN=1

# Generate SwiftUI app
python -m src.iceburg.cli gen-ui "Create a [your app description]" --backends swiftui --out-dir ./my_app
```

### **Method 2: Using ICEBURG Protocol**
```python
import os
os.environ['ICEBURG_ENABLE_VISUAL_GEN'] = '1'
from src.iceburg.protocol import iceberg_protocol

result = iceberg_protocol("Generate a macOS [app type] with [features]")
```

### **Method 3: Direct SwiftUI Generation**
```python
from src.iceburg.agents.visual_architect import VisualArchitect
from unittest.mock import Mock

architect = VisualArchitect()
mock_cfg = Mock()
result = architect.run(mock_cfg, "Create a [app description]", verbose=True)
```
