Core AI/ML Frameworks

Primary frameworks for AI/ML operations, fine-tuning, and model management.

Ollama Primary LLM framework for local model inference. Runs dolphin-mistral, llama3.1:8b, qwen2.5, and other local models. Version: >=0.2.1
Transformers (Hugging Face) Model loading, tokenization, and training. Used in fine-tuning pipeline with AutoModelForCausalLM, AutoTokenizer, TrainingArguments, Trainer.
PEFT Parameter-Efficient Fine-Tuning. LoRA/QLoRA fine-tuning with LoraConfig and get_peft_model for efficient model adaptation.
PyTorch Deep learning backend. Used in quantum ML, reinforcement learning, computer vision, and voice processing. Version: >=2.0.0, <3.0.0
ChromaDB Local vector database for semantic search and knowledge base retrieval. Version: >=0.5.5
Sentence Transformers Semantic embeddings for search and RAG. Used in hybrid search pipeline. Version: >=2.2.0
Rank-BM25 Lexical search (BM25 algorithm) for hybrid search combining BM25 + semantic search. Version: >=0.2.2
MLX Apple Silicon native acceleration for training. NOT INSTALLED - ICEBURG does not include MLX. Mentioned in documentation as recommended option. Status: External framework (not included, install separately if needed)
Unsloth Fast QLoRA training optimized for Apple Silicon. NOT INSTALLED - ICEBURG does not include Unsloth. Has export_for_unsloth() function to export data in Unsloth-compatible format for external use. Status: External framework (not included, install separately if needed)
Llama-Factory Comprehensive fine-tuning framework with multiple quantization options. NOT INSTALLED - ICEBURG does not include Llama-Factory. Has export_for_llama_factory() function to export data in Llama-Factory-compatible format for external use. Status: External framework (not included, install separately if needed)
